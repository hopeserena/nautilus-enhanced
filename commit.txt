feat: The Ultimate Nautilus Enhanced Release v1.0.0

🚀 Features & Visual Overhauls:
- Optimized CJK (Chinese) text rendering and width-calculation for TODO items to prevent UI collision.
- Implemented step-segmentation geometry for cross-hour tasks, allowing them perfectly to hug the spiral.
- Designed smooth Bezier Curve connectors between time slices and text legends.
- Revamped "Completed Tasks" (已完成事项) UI with fading and strikethroughs for better visual satisfaction.
- Refined the central circle layout: Adjusted the red "current-time" pointer to stop clipping the center area, and optically centered the date/year text using physiological visual offset logic.
- Localized the entire Plugin Settings into both English & Chinese, including dynamically swapping text instructions and descriptions.
- Renamed the extension template component to "Nautilus Enhanced" and vastly simplified the onboarding tutorial.

⚡ Performance & Architecture:
- Fixed critical Reagent/React memory leaks! Roam Datalog's polling engine and Regex parsing are now securely memoized under `r/track` hooks, completely neutralizing the CPU-draining minute-tick re-renders.
- General file cleanup and artifact reduction.
